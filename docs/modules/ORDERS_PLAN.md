# خطة وقسم الطلبات — خارطة القسم

## نظرة عامة

قسم **الطلبات** مخصص للعمل الداخلي فقط — تتبع طلبات الشراء (خارجية مع بيتي كاش / داخلية من كاش اليوم) دون ربط محاسبي. يُستخدم لتقارير الأصناف وتتبع عهدة المندوب.

---

## خارطة القسم

```
┌─────────────────────────────────────────────────────────────────┐
│  الطلبات (/orders)                                               │
├─────────────────────────────────────────────────────────────────┤
│  [الطلبات]  [تقارير الأصناف]  [إدارة الأصناف]                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  تبويبة 1: الطلبات                                               │
│  ├── فلتر الشهر (قائمة منسدلة)                                    │

│  ├── قائمة الطلبات (جدول: رقم، تاريخ، نوع، مبلغ، إجراءات)          │

│                                                                  │
│  تبويبة 2: تقارير الأصناف                                        │
│  ├── فلتر الشهر                                                  │
│  ├── جدول: الصنف، الفئة، الكمية، المبلغ، عدد الطلبات             │
│  └── تصدير وطباعة                                                │
│                                                                  │
│  تبويبة 3: إدارة الأصناف                                          │
│  ├── إضافة صنف (اسم، وحدة، فئة، سعر افتراضي)                      │
│  ├── قائمة الأصناف مع تعديل/حذف                                  │
│  └── إدارة الفئات (كرتون، كيلو، حبة، درزن...)                     │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## شرح التبويبات

### تبويبة 1: الطلبات

| العنصر | الوصف |
|--------|-------|
| **الغرض** | إنشاء طلبات جديدة وعرض قائمة الطلبات للشهر المحدد |
| **فلتر الشهر** | قائمة منسدلة لاختيار الشهر (واختيارياً السنة) — كل شهر يُعرض لوحده |
| **طلب جديد** | زر يفتح نافذة إدخال احترافية متكيفة مع الجوال |
| **نوع الطلب** | **خارجي**: مع مبلغ البيتي كاش المحوّل للمندوب. **داخلي**: من كاش المبيعات (بدون بيتي كاش) |
| **قائمة الطلبات** | جدول يعرض: رقم الطلب، التاريخ، النوع، المبلغ الإجمالي، إجراءات (عرض، واتساب، طباعة) |
| **التصدير** | Excel، PDF، طباعة احترافية |

### تبويبة 2: تقارير الأصناف

| العنصر | الوصف |
|--------|-------|
| **الغرض** | عرض ما تم شراؤه في الطلبات حسب الصنف والفئة |
| **البيانات** | الصنف، الفئة، الوحدة، الكمية الإجمالية، المبلغ الإجمالي، عدد الطلبات |
| **الفلاتر** | شهرية فقط |
| **التصدير** | Excel، PDF، طباعة |

### تبويبة 3: إدارة الأصناف

| العنصر | الوصف |
|--------|-------|
| **الغرض** | تعريف الأصناف ووحداتها وفئاتها للاستخدام في الطلبات |
| **الأصناف** | اسم الصنف، الوحدة (كرتون، كيلو، حبة، درزن)، الفئة، السعر الافتراضي (آخر سعر) |
| **الفئات** | تصنيفات الأصناف (مثل: خضروات، لحوم، مشروبات) |
| **الوحدات** | كرتون، كيلو، حبة، درزن — أو ما يُعرَّف في النظام |

## نافذة إدخال الطلب (متكيفة مع الجوال)

### التصميم

- **على الديسك توب**: نافذة منبثقة مركزية (Modal) بعرض مناسب
- **على الجوال**: Bottom Sheet من الأسفل أو نافذة كاملة قابلة للتمرير
- **الحقول**:
  1. تاريخ الطلب
  2. نوع الطلب (خارجي / داخلي)
  3. إذا خارجي: مبلغ البيتي كاش المحوّل للمندوب
  4. الأصناف: اختيار صنف + الكمية + السعر (يُملأ تلقائياً بآخر سعر) + المبلغ
  5. إضافة صنف (زر)
  6. المجموع
  7. ملاحظات
  8. حفظ — إرسال واتساب

### التكيف مع الجوال

- حقول كبيرة للمس
- أزرار واضحة
- تمرير سلس
- إغلاق سهل (زر أو مساحة خارجية)

---

## نوعا الطلبات (مرجع سريع)

| النوع | الوصف |
|-------|-------|
| **خارجي** | طلب من المندوب — يُدخل مبلغ البيتي كاش. يُحسب عهدة المندوب = التحويلات − الطلبات المنفذة |
| **داخلي** | طلب من مدير المطعم — يُدفع من كاش اليوم. لا بيتي كاش |
