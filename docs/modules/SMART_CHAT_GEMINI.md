# المحادثة الذكية + Gemini API

## نظرة عامة

المحادثة الذكية تدعم وضعين:

1. **وضع الكلمات المفتاحية** (افتراضي): مطابقة مباشرة — يعمل بدون أي إعداد إضافي
2. **وضع Gemini** (اختياري): فهم النية عبر Google Gemini — يتطلب مفتاح API

---

## تفعيل Gemini

### 1. الحصول على المفتاح

1. ادخل إلى [Google AI Studio](https://aistudio.google.com/app/apikey)
2. أنشئ مفتاح API جديد
3. انسخ المفتاح

### 2. إضافة المفتاح في Backend

في ملف `backend/.env`:

```env
GEMINI_API_KEY=your-api-key-here
```

**مهم:** لا ترفع ملف `.env` إلى Git. المفتاح يجب أن يبقى سرياً.

### 3. إعادة تشغيل الـ Backend

```bash
cd backend
npm run start:dev
```

## الأمان

| الإجراء | الحالة |
|---------|--------|
| المفتاح في Backend فقط | ✅ لا يُعرض أبداً للواجهة |
| تخزين في `.env` | ✅ خارج الكود |
| Rate limiting | ✅ 30 طلب/دقيقة لكل مستخدم |
| إرسال بيانات مالية لـ Gemini | ❌ لا — يُرسل نص السؤال فقط |
| التحقق من الصلاحيات | ✅ قبل كل استعلام |

## الفترات المدعومة (Phase 1)

| العربية | الإنجليزية |
|---------|------------|
| اليوم | today |
| أمس | yesterday |
| أول أمس / قبل يومين | day before yesterday |
| هذا الأسبوع | this week |
| الأسبوع الماضي | last week |
| هذا الشهر | this month |
| الشهر الماضي | last month |
| السنة | year |

---

## مرادفات إضافية (Phase 1)

- مبيعات: كم حققنا، كم بيعنا، إجمالي المبيعات
- مشتريات: كم اشترينا، إجمالي المشتريات
- مصروفات: كم صرفنا، إجمالي المصروفات
- مساعدة: ماذا تعرف، ما الذي، أسئلة

## التدفق التقني

```
المستخدم: "كم حققنا من مبيعات أول أمس؟"
    │
    ├─ [Gemini متاح] ──► Gemini: { intent: "sales", period: "day_before_yesterday" }
    │                         │
    │                         └─► Handler (sales) ──► Ledger ──► "مبيعات أول أمس: X ﷼"
    │
    └─ [Gemini غير متاح] ──► مطابقة كلمات: "مبيعات" + "أول أمس" ──► نفس النتيجة
```
